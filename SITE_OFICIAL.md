# Site oficial da Sartec Papelaria

O site oficial da Sartec Papelaria não é mais mantido dentro deste repositório.

Este repositório, `agente-triagem-crm-sartec`, deve ser usado para o CRM, o painel de atendimento, o agente WhatsApp, o webhook e as APIs operacionais da Sartec.

O site institucional oficial está separado em outro repositório:

`Suertesoy/sartecpapelaria`

O projeto Vercel responsável pelo site oficial é:

`sartec`

O domínio oficial do site é:

`https://sartecpapelaria.com.br/`

Antes de fazer qualquer ajuste visual no site, abra o projeto/repositório correto do site oficial. Não edite a pasta do CRM/agente esperando alterar o site em produção.

Mapa atual:

CRM, painel, agente e webhook:
`Suertesoy/agente-triagem-crm-sartec`

Site oficial:
`Suertesoy/sartecpapelaria`

Esta separação existe para evitar mistura entre produto operacional de atendimento e site institucional.
